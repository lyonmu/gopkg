# gopkg

一个 Go 语言工具包，收集和整理在开发工作中常用的工具函数。

## 项目结构

```
gopkg/
├── pinyin/    # 中文拼音转换工具
├── docs/      # 文档
└── ...        # 持续收集中...
```

## 模块

| 模块 | 说明 |
|------|------|
| [pinyin](pinyin/README.md) | 中文拼音转换工具，支持首字母、完整拼音、驼峰格式及中文检测 |

## 安装

```bash
go get github.com/lyonmu/gopkg
```
